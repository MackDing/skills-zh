# Skills 中文版（skills-zh）

这是 [`mattpocock/skills`](https://github.com/mattpocock/skills) 的中文化分支 / 镜像仓库，目标是把原仓库里对工程实践、Agent 协作、TDD、诊断、PRD、问题拆解等实用 skills 逐步整理成更适合中文开发者阅读和使用的版本。

> 当前阶段：初始化中文仓库说明。
> 还没有完成全量逐条汉化，后续会按优先级逐步推进。

## 仓库目标

- 保留原仓库的技能结构与方法论
- 提供中文 README / 使用说明 / 导航
- 逐步把高价值 skills 翻译为中文版本
- 在必要处补充更适合中文开发者语境的解释

## 上游仓库

- 上游：https://github.com/mattpocock/skills
- 当前仓库： https://github.com/MackDing/skills-zh

如果你更想看英文原版，直接使用上游即可。

## 使用建议

如果你准备自己继续汉化，建议采用这套策略：

1. 先翻译顶层说明文档
2. 再翻译高频 skills
3. 尽量保留原始 skill 名称，避免安装路径和引用关系被破坏
4. 中文说明可放在原文旁边，或逐步替换为双语版本
